# MiniShop - Simple E-Commerce Demo

Features:
- Navigation: Home, Products, Orders, Wishlist, Contact, About
- Registration & Login with basic validation (stored in localStorage)
- Add to wishlist, place orders (localStorage)
- Contact form (stores messages), Reviews

1. Put `index.html`, `styles.css`, `script.js` in same folder.
2. Open `index.html` in browser.
